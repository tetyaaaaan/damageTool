## 重要な仕様

アモスの弓は、常時発動する通常・重撃ダメージ補正と、矢の飛翔時間に応じた追加補正の二つを持つ。両方が表示されるのは重複ではなく仕様である。結果明細では `effectLabel` により「常時発動する基礎効果」と「飛翔時間による追加効果」を区別し、飛翔時間は既定値 0 段とする。

申鶴・スカークで見えていた項目はロード警告ではなく、命ノ星座のリソース層数が未入力であることを示す計算候補だった。候補データは監査用に保持するが、同じ内容を全ダメージ明細へ反復しない。必要な入力だけを「追加入力待ち」として一括表示する。

専用参照入力は、海染硨磲などの記録治療量、または申鶴・閑雲型の別キャラクター由来補正で提供者 HP・攻撃力・防御力・元素熟知を参照するために使う。選択中の効果が要求する欄だけを表示する。

## 検証

ローカルと CI は次の一つのコマンドで同じ検証を実行する。

```text
node scripts/verifyGenshinCalc.cjs
```

この検証は P0/P1 が 0 件であること、ゴールデン値が一致すること、原神 JSON 計算の全 Node テストが通ることを確認する。P2 は安全な共通式へ落とせるものから継続対応し、誤った汎用化は行わない。
