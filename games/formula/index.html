計算式

各ツールで使っているダメージ計算式と、補正項目の考え方をまとめています。

原神

ダメージ = 基礎ダメージ * ダメージバフ補正 * 会心ダメージ補正 * 元素反応補正 * 敵防御補正 * 敵元素耐性補正

  • 基礎ダメージ = 攻撃力 * 天賦倍率 * 特殊乗算 + 実数ダメージ加算
  • ダメージバフ補正 = 1 + ダメージバフ%の合計値
  • 会心ダメージ補正 = 1 + 会心ダメージ%の合計値
  • 蒸発・溶解は反応倍率と元素熟知補正を反映します。
  • 激化系はLv90固有値を前提に固定値加算として扱います。

崩壊スターレイル

ダメージ = 基礎ダメージ * ダメージバフ補正 * 会心ダメージ補正 * 被ダメージ補正 * 靭性補正 * 敵防御補正 * 属性耐性補正

  • 基礎ダメージ = 攻撃力 * 倍率
  • 攻撃力 = 基礎攻撃力 * (1 + 攻撃%) + 攻撃実数値
  • 敵防御補正ではキャラレベルと敵レベルに固定レベル補正を加えます。
  • 属性耐性補正は弱点有無と耐性デバフをもとに計算します。

注意点

本ツールは公開情報と一般的な計算式をもとにした目安計算です。キャラクター固有効果、条件付きバフ、敵固有ギミックなど、すべての状況を完全に再現するものではありません。
